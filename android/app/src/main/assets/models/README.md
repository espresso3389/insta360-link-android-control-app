# YOLOv8n-face model placement

Put the face model here with one of these filenames:
- `yolov8n-face.tflite` (recommended)
- `yolov8n_face.tflite`

The app loads this model for Android Camera2 face detection and gimbal tracking.

# EdgeFace-S embedding model placement

Put the face embedding model here with one of these filenames:
- `edgeface_s.tflite` (recommended)
- `edgeface-s.tflite`

The app uses this model to compute face embeddings for local matching.
